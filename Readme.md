#  "Cloud hosting"

* Author: Anatoliy Vershinin


    Для запуска сборки нужно установить 
    необходимые пакеты. 
    
    Команды:

      Установить все зависимости

          npm i
          bower i

      Запустить dev-сборку проекта и сервер

          npm start

      Запустить build-сборку:

          npm run build
          
      Запустить проверку codestyle:
        
          npm run codestyle
